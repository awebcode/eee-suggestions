Section A: Diodes

Q1. Define a Diode. 2 Marks

Definition: A diode is a two-terminal semiconductor device that allows current to flow in only one direction (unidirectional conduction). It acts as an electronic check valve.

Q2. Types of Diodes. 3 Marks

  • PN Junction Diode: Basic diode for rectification
  • Zener Diode: Used for voltage regulation (reverse bias)
  • LED (Light Emitting Diode): Emits light when forward biased
  • Photodiode: Converts light to electrical current
  • Schottky Diode: Fast switching with low forward voltage drop
  • Tunnel Diode: Used in high-frequency applications
  • Varactor Diode: Voltage-controlled capacitor

Q3. Diode Characteristics (V-I Curve). 3 Marks

Three Regions:

  • Forward Bias: When V > 0.7V (Si) or 0.3V (Ge), diode conducts heavily
  • Reverse Bias: Small leakage current (ÂµA range)
  • Breakdown Region: Reverse voltage exceeds breakdown voltage, current increases sharply

Q4. Applications of Diodes. 2 Marks

  • Rectification: AC to DC conversion
  • Voltage Regulation: Using Zener diodes
  • Signal Clipping & Clamping: Signal shaping
  • Protection Circuits: Reverse polarity protection
  • Demodulation: Extracting signals from modulated waves
  • Switching: Fast switching applications

Section B: Transistors (BJT)

Q5. Define BJT (Bipolar Junction Transistor) with Symbol. 3 Marks

Definition: A BJT is a three-terminal semiconductor device consisting of two PN junctions. It has three regions: Emitter (E), Base (B), and Collector (C). Current flow involves both electrons and holes (bipolar).

Note: Arrow on emitter indicates conventional current direction. NPN: arrow points OUT, PNP: arrow points IN.

Q8. Types of Transistors. 2 Marks

  • BJT (Bipolar Junction Transistor):
    • NPN Transistor
    • PNP Transistor
  • FET (Field Effect Transistor):
    • JFET (Junction FET) - N-channel, P-channel
    • MOSFET (Metal Oxide Semiconductor FET)
      • Enhancement Mode (E-MOSFET)
      • Depletion Mode (D-MOSFET)

Q9. Characteristics of BJT. 3 Marks

Key Characteristics:

  • Current-controlled device: IC depends on IB
  • Low input impedance: Few hundred ohms
  • High output impedance: Several kÎ©
  • Temperature sensitive: Performance varies with temperature
  • Power consumption: Higher than FET
  • Switching speed: Moderate (slower than MOSFET)
  • Gain (Î²): Typically 50-200

Three Main Characteristics:

  • Input Characteristics: IB vs VBE (VCE constant)
  • Output Characteristics: IC vs VCE (IB constant)
  • Transfer Characteristics: IC vs IB (VCE constant)

Q10. Applications of Transistors. 2 Marks

  • Amplification: Audio, RF, and power amplifiers
  • Switching: Digital circuits, logic gates
  • Oscillators: Signal generation
  • Voltage Regulation: Power supplies
  • Signal Processing: Modulation, demodulation
  • Motor Control: Speed and direction control

Section C: MOSFET

Q11. Define MOSFET with Symbol. 3 Marks

Definition: MOSFET (Metal Oxide Semiconductor Field Effect Transistor) is a voltage-controlled three-terminal device with Gate (G), Drain (D), and Source (S). Current flow is controlled by electric field.

Note: Enhancement mode has broken channel (dashed lines). Arrow direction: N-channel points toward channel, P-channel points away.

Q18. Biasing Techniques. 2 Marks

  • Fixed Bias: Simple but temperature unstable
  • Voltage Divider Bias: Most stable, widely used
  • Emitter Bias: Good stability with negative supply
  • Collector Feedback Bias: Moderate stability

Q19. Early Effect in BJT. 2 Marks

As VCE increases, collector-base depletion width increases, reducing effective base width. This causes IC to increase slightly with VCE even in active region.

Result: Output characteristics show slight upward slope instead of being horizontal.

Q20. Advantages and Disadvantages. 3 Marks

BJT Advantages:

  • High current handling capacity
  • Better for analog/linear applications
  • Lower cost
  • Higher transconductance

BJT Disadvantages:

  • Lower input impedance
  • Temperature sensitive
  • Slower switching speed
  • Requires continuous base current
